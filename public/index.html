littlefoot.js

itty bitty footnotes

Fork me on GitLab

Littlefoot transforms your footnotes into inline clickable references.1 It does this by parsing the DOM of your page and converting all footnotes into inline notes. This means readers with JavaScript disabled will still be able to read your footnotes.2

Unlike its older cousin, Bigfoot, Littlefoot does not require jQuery. Littlefoot is less than 100 lines of pure, vanilla JavaScript.

Furthermore, Littlefoot's inline notes have a small footprint. This makes them easy to customize with CSS.3

Littlefoot also allows you to use full HTML markup in your footnotes, and it'll automatically present a scrollwheel to accommodate lengthy notes.4

Usage

Include littlefoot.js at the end of your page:

<!doctype html>
<html>
<body>
	...
	<script src="littlefoot.js"></script>
</body>
</html>

Then use stomp() to shake up your DOM and move those footnotes inline!

<script>new LittleFoot().stomp()</script>

That's all there is to it!

Littlefoot is preconfigured to use MultiMarkdown format footnotes. But you can change this. Pass options to the LittleFoot constructor:

Option Default Value Description
footnoteContainer .footnotes Selector of the DOM node that contains your footnotes (ie, the notes at the end of your document).
footnoteSelector li Selector of the DOM node that contains a single footnote.
footnotePrefix #fn: Selector of the DOM node that contains the body of a footnote. The footnote number will be appended to this option, so #fn: will become #fn:1, #fn:2, etc.
inlinePrefix #fnref: Selector of the DOM node that contains the inline reference to a footnote. The footnote number will be appended to this option, so #fnref: will become #fnref:1, #fnref:2, etc.
inlineDepth 1 How "deep" into the DOM tree inline footnote references are found. For instance, most text processors will place inline references in a <sup> element, giving the default depth 1.
removeInner [".reversefootnote"] Array of selectors to remove from footnotes before moving them inline.

For example, I use the following code to use Littlefoot with the Kirby Footnotes plugin for Kirby CMS:

new LittleFoot({
	footnoteContainer: "#footnotes",
	footnotePrefix: "fn-",
	inlinePrefix: "fnref-",
	removeInner: [".footnotereverse"]
}).stomp()
